El Comienzo

Aquí puedes escribir cómo empezó todo. Este diseño coloca el texto a la izquierda en PC y la imagen a la derecha.

En celulares, la imagen se pondrá automáticamente arriba para que se vea bien.

Crecimiento

En esta sección el orden está invertido. La imagen domina el lado izquierdo.

Es ideal para mantener el ritmo visual y que el espectador no se aburra de ver siempre lo mismo.

Un Momento Clave

A veces una imagen vale más que mil palabras.

Este estilo usa la foto como fondo con un filtro oscuro para que puedas leer el texto encima.
